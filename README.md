# עדכון חדשות יומי

תקציר חדשות ישראלי יומי בעברית, מותאם אישית — בנוי כאתר סטטי (PWA) שאפשר להתקין על מסך הבית בטלפון.

## מבנה הפרויקט

- `site/` — האתר הסטטי שמתפרסם ב-GitHub Pages.
- `data/` — קבצי הנתונים: `latest.json` (התקציר של היום), `preferences.json` (תחומי עניין), `topic_catalog.json` (רשימת נושאים לבחירה), `archive/` (היסטוריה).
- `automation/` — הסקריפטים שרצים פעם ביום על המחשב המקומי: שולפים כותרות, מסכמים עם Claude (מודל זול), ודוחפים ל-GitHub.
- `scripts/` — התקנת תזמון (`launchd`) והרצה/בדיקה ידנית.
- `.github/workflows/` — פריסה אוטומטית ל-GitHub Pages בכל push.

## פיתוח מקומי

```bash
./scripts/dev_serve.sh
```

פותח שרת מקומי שמדמה בדיוק את מבנה הנתיבים של הפריסה בפרודקשן (`data/latest.json` יחסית ל-`site/`).

## הרצה ידנית של האוטומציה

```bash
./scripts/run_now.sh
```

## התקנת ריצה יומית אוטומטית

```bash
./scripts/install_launchd.sh
```

מתזמן הרצה יומית סביב 07:00 (משלים ריצה שהוחמצה בהתעוררות הבאה של המחשב אם היה כבוי/ישן).
